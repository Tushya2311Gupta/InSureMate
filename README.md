# InSureMate

A full-fledged insurance management platform for companies.

## Features
- Policy management
- Claims processing
- AI recommendations
- Web & mobile apps

## Tech Stack
- **Frontend:** React.js, React Native
- **Backend:** Node.js, Python
- **Database:** MongoDB
- **DevOps:** Docker

## Setup
1. Clone the repo
2. Install dependencies: `npm install` (frontend/backend)
3. Set up `.env` (see `.env.example`)
4. Run with Docker: `docker-compose up`
